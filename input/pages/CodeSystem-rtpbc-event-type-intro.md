This code system establishes the event types used in Real-time Pharmacy Benefit Check message headers.

<p class="stu-note"> This version of the IG adds two codes to support the  new non-personally-identifiable request and response flow:
- rtpbc-request-non-phi, "RTPBC Non-PHI Request"
- rtpbc-response-non-phi, "RTPBC Non-PHI Response"
